# 鸭鸭照使用帮助文档

## 使用方法

使用 **哎鸭** / **哎呀** / **aiya** 主动触发，支持主动进行 **早上好/中午好/晚上好/晚安鸭** 问好

如果需要使用问好功能，需要进行配置操作，参考下发的配置指南
